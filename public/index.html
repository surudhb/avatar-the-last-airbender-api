Avatar: The Last Airbender API

"All in a day's work for the Avatar!" - Aang

GET /api/v1

  • /bending : Returns a JSON array of all bending types in Avatar.
  • /bending/firebending : Returns a single JSON associated with firebending.

    • Response schema: 

    {
"name",
"description",
"inspiration",
"pioneers": [],
"notableUsers": [],
"techniques": [],
"advancedTechniques": [],
"specializations": [],
"strengths": [],
"weaknesses": []
}

  • /episodes : Returns a JSON array of all episodes of Avatar.
  • /episodes/id/203 : Returns a single JSON object for Season 2 Episode 3
  • /episodes/title/Avatar Day : Returns a single JSON object for the episode titled Avatar Day

    • Response schema: 

    {
"name",
"id",
"book",
"synopsis",
"directedBy",
"writtenBy": [],
"animatedBy",
"firstAired"
}
  • /locations : Returns a JSON array of all locations in Avatar.
  • /locations/name/Omashu : Returns a single JSON object about the location Omashu

    • Response schema: 

    {
"name",
"type",
"capital",
"nation",
"location",
"description",
"governmentType",
"legalAuthority",
"spiritualAuthority",
"notableLeaders" : [],
"notableMembers": [],
"currency",
"firstAppearance"
}
  • /organizations : Returns a JSON array of all organizations/teams in Avatar.
  • /organizations/name/Kyoshi Warriors : Returns a single JSON object about the Kyoshi Warriors

    • Response schema: 

    {
"name",
"description",
"notableMembers": [],
"founders": [],
"leaders": [],
"affiliations": [],
"headquarters",
"firstAppearance"
}

FAQ

Is this free to use? YES.
How often will new features be added? Not fast or soon enough :). This made me realize cleaning up data is bitter work.
Is there a limit? Yes! The current limit is set to 100 requests/hour.
Why did you make this? I love Avatar and wanted to learn to make an API.
Where did you get all this data from? All this data is straight from the Avatar fandom wiki (with slight modifications). Check them out here.
Who's your favourite character? Uncle Iroh <3.
Kataang or Zutara? As much as I love Zutara, I think people severely underestimate what it took to get Kataang. Kataang for sure.
Are you planning to add The Legend of Korra? Yes! But first I will be adding whatever I can to finalize the core show's API.
I really wanted information on the characters, but you haven't done that yet :( I promise I'm working hard on it! In the meantime check out this other ATLA API made by a fellow member of the Boomeraang Squad here.
