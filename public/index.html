Avatar: The Last Airbender API

"All in a day's work for the Avatar!" -Aang

GET /api/v1

  • /bending : Returns a JSON array of all bending types in Avatar.
  • /bending/firebending : Returns a single JSON associated with firebending.
  • /episodes : Returns a JSON array of all episodes of Avatar.
  • /episodes/id/203 : Returns a single JSON object for Season 2 Episode 3
  • /episodes/title/Avatar Day : Returns a single JSON object for the episode titled Avatar Day
  • /locations : Returns a JSON array of all locations in Avatar.
  • /locations/name/Omashu : Returns a single JSON object about the location Omashu
  • /organizations : Returns a JSON array of all organizations/teams in Avatar.
  • /organizations/name/Kyoshi Warriors : Returns a single JSON object about the Kyoshi Warriors
